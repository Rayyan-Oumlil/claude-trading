# Knowledge

My growing reference for trading concepts. Every term I learn, every concept I internalize, gets a short entry here.

| File | Purpose |
|------|---------|
| [glossary.md](glossary.md) | One-paragraph definitions of every term I need |
| [concepts/](concepts/) | Deeper notes on non-trivial topics |

## Rules

- When Claude introduces a new term I don't know, I add it to `glossary.md` that session.
- When a concept needs more than a paragraph (e.g. "what is Sharpe really measuring"), it gets its own file in `concepts/`.
- When my understanding of a term changes, I update the entry (don't create a second one).

## Why this exists

Most trading mistakes are caused by using a term without understanding it. Writing my own definition forces comprehension. Having my own dictionary means I'm not constantly re-learning the same 50 words.
