# Prompt — Daily Plan (morning session opener)

Use this first thing when I sit down to trade or work on the system.

---

```
Good morning. Before we start:

1. Read the latest entry in `journal/`. Summarize any open threads.
2. Check the active strategy list in `CLAUDE.md` section 6. For each,
   state stage (research/backtest/paper/live) and whether it has an
   open question.
3. Via MCP, pull:
   - SPY and QQQ overnight + pre-market move
   - BTC/ETH 24h move
   - VIX
   - Any earnings / macro event today for my watchlist
4. Flag anything in today's calendar that would invalidate a running
   paper/live strategy (FOMC, CPI, earnings on a held name).
5. Propose one focus for the session, picking from: research, design,
   backtest, paper review, journaling.

Wait for my response before doing anything else.
```

---

## Why this prompt

- Starts with context (principle 5) before acting.
- Uses MCP for live data (principle 4).
- Proposes one focus — tight scope prevents the "do a little of everything" session that produces nothing.
