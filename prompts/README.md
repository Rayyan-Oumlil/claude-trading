# Prompts

Reusable prompt templates. Each one is a starting point — customize for the specific ask. These encode the seven principles so I don't have to restate them every session.

| File | Use when |
|------|----------|
| [research-stock.md](research-stock.md) | Starting research on a new ticker or market |
| [design-strategy.md](design-strategy.md) | Turning a vague idea into a spec Claude can build |
| [backtest-review.md](backtest-review.md) | Reading a freshly-run backtest |
| [daily-plan.md](daily-plan.md) | First thing in the morning |
| [post-trade-review.md](post-trade-review.md) | End of a paper or live trading session |
| [kill-the-idea.md](kill-the-idea.md) | Stress-testing a strategy before committing |
| [routine-premarket.md](routine-premarket.md) | 6am weekday pre-market routine |
| [routine-market-open.md](routine-market-open.md) | 8:30am weekday market-open routine |
| [routine-midday.md](routine-midday.md) | Noon weekday risk check routine |
| [routine-close.md](routine-close.md) | 3pm weekday close routine |
| [routine-weekly-review.md](routine-weekly-review.md) | Friday 4pm weekly review routine |

## How to use

Copy the prompt, fill in the `<placeholders>`, paste into Claude Code. Or reference the file and say "run the template at `prompts/design-strategy.md` for a mean-reversion idea on SPY."
